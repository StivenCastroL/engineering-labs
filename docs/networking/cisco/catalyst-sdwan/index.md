# Cisco Catalyst SD-WAN

Laboratorios de Cisco Catalyst SD-WAN (anteriormente Cisco SD-WAN / Viptela).

Estos laboratorios cubren desde el despliegue inicial del overlay hasta políticas avanzadas, seguridad y optimización de tráfico.

## Entorno de laboratorio

| Componente | Detalle |
|------------|---------|
| Plataforma | EVE-NG |
| Imágenes | Cisco SD-WAN Manager, Validator, Controller, WAN Edge (C8000v) |
| Versión | 20.x |
