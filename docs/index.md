# Engineering Labs

Laboratorio personal de experimentación técnica en networking, infraestructura, seguridad e inteligencia artificial.

Cada laboratorio sigue una metodología de documentación estructurada que permite reproducir los experimentos, consultar resultados durante proyectos reales, y construir una cadena de aprendizaje progresiva entre labs.
