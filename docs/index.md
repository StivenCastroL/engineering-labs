# Engineering Labs

Welcome to my engineering laboratory environment.

This project is used to **learn, test and validate real technologies** related to:

- Networking
- Infrastructure
- Security
- Artificial Intelligence

---

## Methodology

Each laboratory follows a structured engineering methodology:

1. Objective
2. Technical context
3. Architecture
4. Hypothesis
5. Procedure
6. Validation
7. Results
8. Troubleshooting
9. Lessons learned
10. Next experiments

---

## Lab Categories

### Networking

Labs related to routing, switching, SD-WAN and network architecture.

### Infrastructure

Labs related to virtualization, containers and infrastructure platforms.

### Security

Security engineering labs including SIEM, EDR and Zero Trust.

### Artificial Intelligence

Experiments with AI agents, automation and local LLMs.
